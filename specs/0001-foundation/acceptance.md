# Spec 0001 — Foundation acceptance

## What "v0 done" means

Spec 0001 is closed when:

1. The three PRs in `tasks.md` are merged.
2. `uv pip install -e .[dev]` succeeds.
3. `python -m policy_replay --help` prints the CLI surface.
4. The PJM Dec 2025 capacity-reform rule replays end to end against
   the checked-in 2018-2023 history fixture.
5. `reports/pjm-dec2025-replay/report.md` exists, was generated by
   the simulator, and passes voice_lint plus the no-advocacy check.
6. The MOPR calibration backtest runs and reports a numeric miss
   under the tolerance set in
   `decisions/DEC-PR-002-calibration-tolerance.md`.
7. All local gates pass.

## Commands to run

```bash
uv pip install -e .[dev]
python -m policy_replay --help

uv run pytest
python scripts/voice_lint.py
python scripts/validate_schemas.py

python -m policy_replay simulate \
  --rule rules/pjm-dec2025-capacity-reform.yaml \
  --history data/pjm_rpm_2018_2023.parquet \
  --trials 1000 \
  --seed 42 \
  --out reports/pjm-dec2025-replay/

python eval/sanity_bounds.py reports/pjm-dec2025-replay/replay_result.json
python eval/calibration_against_past_changes.py
```

## Gates

| Gate | Source | Blocks merge when |
|---|---|---|
| pytest | `tests/` | any test fails |
| determinism | `tests/test_replay_determinism.py` | same seed produces different output |
| voice_lint | `scripts/voice_lint.py` | banned term or advocacy verb in `reports/` |
| schemas | `scripts/validate_schemas.py` | any rule, history, or replay record fails |
| sanity bounds | `eval/sanity_bounds.py` | counterfactual outcomes outside plausible range |
| calibration | `eval/calibration_against_past_changes.py` | MOPR backtest misses beyond tolerance |

## What v0 explicitly does NOT include

- Nodal market-clearing. Zonal / LDA-level only.
- Generation-asset-level dispatch.
- Live ISO ingestion.
- Equity / generation-asset valuation outputs.
- Policy recommendations.
- A web frontend or hosted SaaS.
